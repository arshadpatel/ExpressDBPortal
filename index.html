Description

Express DB Portal" is a web application built using the Express framework that serves as a portal for managing a database. The project integrates authentication, database connectivity, and different views to provide users with a convenient way to interact with the database. Here's a brief description of the project:

User Authentication:


The project includes a login system where users can authenticate themselves using a username and password.

User credentials are stored and validated against a MySQL database table named auth.

Upon successful authentication, users are granted access to specific functionalities based on their roles.



Database Integration:


The project establishes a connection to a MySQL database using the mysql module.

This connection allows the application to interact with the database, perform queries, and retrieve data.

The database connection details, such as host, username, password, and database name, are configured in the code.



Dynamic Pages and Views:


The project utilizes the Handlebars templating engine to render dynamic views and templates.

Views are rendered based on different routes, allowing users to access specific functionalities and information.

The project includes multiple views, such as the index page, customer page, and admin page, each with their own purpose and content.



Functionality:


The index page serves as the entry point of the application, displaying a login form for users to authenticate themselves.

Once logged in, users are redirected to either the admin page or the customer page based on their assigned role.

The admin page provides an overview of data retrieved from the database, such as quantity, weight, and box count.

The customer page allows users to submit data to the database using a form, capturing information like date, company, owner, and item details.

Submitted data from the customer page is stored in the customer table of the database for further processing.



Styling and Static Assets:


The project includes static HTML, CSS, and JavaScript files that are served to the client..

Static assets, such as stylesheets and scripts, are stored in the "public" directory and are accessible to the user interface..

The application uses different stylesheets and scripts based on the views being rendered, providing a visually appealing and interactive user experience.



"Express DB Portal" simplifies the management of a database by providing a user-friendly interface for authentication, data submission, and data retrieval. It empowers users with different roles to perform their respective tasks efficiently and securely while leveraging the power of the Express framework and MySQL database integration.
